That's all :)

## Source files
1. main.cppm
2. Game.ixx
3. Player.ixx

## Source module tests files

## Project settings
C++20 standart, MVSC.  
The binaries will be builded in $(ProjectDir)\bin\$(Platform)\$(Configuration)\  
The intermidates will be builded in $(ProjectDir)\Intermidate\$(Platform)\$(Configuration)\
